Heap

  • Constant time retrieval, log time updation
  • Priority queues are usually implemented using heaps ʳ
    • So much so that priority queues are sometimes just called heaps.

Binary heap is a complete binary tree

Max-heap:

  • Value of parent ≥ value of children

Min-heap:

  • Min element is always at root
  • No parent node is greater than its children

Min-max-heap:

  • Combines advantages of both min-heap and max-heap
  • Useful for implementing double-ended priority queues
