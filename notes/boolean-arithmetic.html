Boolean arithmetic

Boolean

Named after George Boole.

Boolean algebra properties

Associativity

  • a + (b+c) = (a+b) + c
  • a(bc) = (ab)c

Absorption

  • a + ab = a
  • a(a + b) = a

de Morgan's laws

  • (ab)' = a' + b'
  • (a+b)' = a'b'

Identity

  • a + 0 = a
  • a.1 = a

Commutativity

  • a.b = b.a
  • a + b = b + a

Distributivity

  • a(b + c) = ab + ac
  • a + bc = (a + b).(a + c)

Idempotency

  • a.a = a
  • a + a = a

Involution

aka 'double negation'.

  • (a')' = a

Universal gates

  • Can be used to realize any other gate.
  • NAND, NOR

NAND

  • NOT: (A.A)' → 1 NAND gate
  • AND: (AB)'' → 2 NAND gates
  • OR: (A+B)'' = (A'B')' → 3 NAND gates

NOR

  • NOT: (A+A)' → 1 NOR gate
  • AND: (AB)'' = (A'+B')' → 3 NOR gates
  • OR: (A+B)'' → 2 NOR gates
