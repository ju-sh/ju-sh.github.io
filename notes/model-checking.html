Property Atomic propositions

Trace

Trace of an execution is an infinite word over (ie, over the powerset of atomic propositions). ie, Trace(Execution) = {x | x ∈ 2ᴬᴾ}

Traces

Traces of a transition system is the trace of all executions of that TS. ie, Traces(TS) = {Trace(σ) | σ is a execution of TS}

Traces of a TS describe the behaviour of the TS with respect to a set of atomic propositions.

Transition system (TS)

A tuple

(S, Act, →, I, AP, L)

where

  • S: set of states
  • Act: set of actions
  • →: transition function
    • → ⊆ S⨯Act⨯S (old state + action = new state)
  • I: set of initial states
    • I ⊆ S
  • AP: set of atomic propositions
    • Chosen based on the charecteristics of interest (eg: register states may not be of interest)
  • L: labeling function
    • S → 2ᴬᴾ
    • L(S) set of atomic propositions which are satisfied at state S

Finite TS: TS whose S, Act, AP are all finite sets.

Non-deterministic transition if we can't know for sure which path is followed.

Run

  • Describes a possible behaviour of a transition system.
  • Execution: A finite prefix of a run.
