Order of function calls made in TCP

Server:

  1. socket: create a socket
  2. bind: bind the socket with an ip address
  3. listen: await client requests
  4. accept: accept client request
  5. recv / read: read data from client
  6. send / write: give data to client
  7. close: close connection

Client:

  1. socket: create a socket
  2. connect: trigger TCP handshake (3-way: SYN, SYN-ACK, ACK)
  3. send / write: give data to server
  4. recv / read: get server response
  5. close: close connection
