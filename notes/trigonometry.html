Definitions

  • sine: sinθ = opposite / hypotenuse

  • cosine: cosθ = adjacent / hypotenuse

  • tangent: tanθ = opposite / adjacent

  • cosec: cosecθ = 1/sinθ

  • sec: secθ = 1/cosθ

  • cotangent: cotθ = 1/tanθ

  • arcsin = sin⁻¹

Periodicity

Functions with period 2π:

| Function | θ ± (π/2) | +π      |
|----------+-----------+---------|
| sin      | ± cosθ    | -sinθ   |
| cos      | ∓ sinθ    | -cosθ   |
| cosec    | ± secθ    | -cosecθ |
| sec      | ∓ cosecθ  | -secθ   |

Functions with period π:

| Function | θ ± (π/4) | +(π/2) |
|----------+-----------+--------|
|          | tanθ ± 1  |        |
| tan      | ───────── | ─cotθ  |
|          | 1 ∓ tanθ  |        |
|----------+-----------+--------|
|          | cotθ ∓ 1  |        |
| cot      | ───────── | ─tanθ  |
|          | 1 ± cotθ  |        |
