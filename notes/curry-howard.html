Type theory

Originally made to get around Russell's paradox which couldn't be dealt with axiomatic set theory.

Formal language: Each term is derivable from the rules.

Soundness and completeness

Completeness: Γ ⊢ φ

  • Every provable proposition can be derived from Γ

Soundness: Γ ⊨ φ

  • If a proposition is modelled by φ, then it is true.
