Logic

L11

Γ ⊢ α means we can derive α from Γ.

Proofs are made relative to a proof system.

A Hilbert style proof system consists of:

  • Axioms: set of formulas
  • Inference rules: set of rules. Relations between from premises to conclusions.

Usually only axiom schemas and inference rule schemas are mentioned. Substitute as needed.

The notion of proof

L15

  • Γ ⊢ α: α is derivable from Γ.
  • Γ ⊨ α: α is provable from Γ.
    • Γ ⊨ α means α is indeed a logical consequence of Γ
  • Soundness: Γ ⊢ α means Γ ⊨ α
    • If derivable, it is correct.
  • Completeness: Γ ⊨ α means Γ ⊢ α
    • If correct, it is derivable.

To show soundness, all we got to do is prove that

  • all axioms are valid
  • all inference rules preserve validity.
Consistency

A set of formulas is consistent if there is no formula α such that Γ ⊢ α and Γ ⊢ ¬α.

  • ie, no contradictions.

Proof of completeness is same as having proof that every consistent set of formulas is SAT.

Hilbert system

Only one inference rule: Modus ponens.

(p → q), p ⊢ q
