UTLC

Types (we are considering functions as values. Kind of like in Church numerals):

τ :=  τ → τ   # Function type

Values:

v := λx:τ. t

Context:

Γ := ∅       # Empty context
   | Γ, x:τ  # Add a var to context

Typing rules:

  x:τ ∈ Γ 
─────────── (Var)
  Γ ⊢ x:τ


 Γ ⊢ t: τ1 → τ2      Γ ⊢ t1: τ1
───────────────────────────────── (App)
          Γ ⊢ (t t1): τ2 



       Γ, x: τ1 ⊢ t: τ2
───────────────────────────────── (Abs)
     Γ ⊢ (λx:τ1. t): τ1 → τ2

Evaluation rules:

           t1: τ
───────────────────────────── (AppAbs)
  (λx:τ. t) t1  →  t[t1/x]



        t1 → t1' 
─────────────────────── (AppL)
  t1 t2   →  t1'  t2



       t2 → t2' 
───────────────────── (AppR)
  t1 t2   →  t1 t2'
