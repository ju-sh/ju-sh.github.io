SystemVerilog Assertions

  • Boolean negation: not p or ! p ??
  • p and q: p and q should be true, but both need not end at same time
  • p intersect q: Concurrent and
    • also p && q ??
  • p ##N q: check if p holds, then check if q holds N cycles later
    • Eg: p ##2 q
  • p [*N]: p is repeated N times
    • Eg: p [*3]
  • Implication:
    • Overlapping (p |-> q)
      • If p is true, then q must hold on the same cycle
    • Non-overlapping (p |=> q)
      • If p is true, then q must hold on the next cycle
      • Equivalent to something like p ##1 (true |-> q)

Terms

  • Assertion
  • Assumption
  • Cover point
    • Something that is only occassionally true
    • Kind of like coverage in tests
