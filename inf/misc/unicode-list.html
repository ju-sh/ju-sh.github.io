Info: Unicode v15

  • 17 planes:
    • Indexed from 0 to 16
    • Each plane has 2¹⁶ code points
  • 327 blocks
    • Sub-divisions of plains
    • Contiguous code points
    • No fixed size
  • High surrogate code points: D800-DBFF
  • Low surrogate code points: DC00-DFFF

Planes

U+ppxxxx: where pp denotes the plane.
