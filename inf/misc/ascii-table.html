  • ^[: Escape (bash: $'\u1b')
  • ^: Ctrl
  • C-j and Enter key are indistinguishable
  • Escape and Alt keys are indistinguishable
  • Tilde indicate end of code (I guess)
  • If IFS variable is at least declared, we can distinguish between space and enter in bash
