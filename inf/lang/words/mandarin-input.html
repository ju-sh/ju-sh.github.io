Chinese computer input

Some of the input methods

  • Wubi
    • For simplified Chinese. Not very good for traditional characters. ʳ
    • Wubizixing:
      • wǔ (5) bǐ (stroke) zì (letter) xíng shūrùfǎ
    • Wubi86: Most common input method in mainland China?
    • 5-stroke
  • Cangjie
    • Developed in Taiwan
    • Supports simplified and traditional letters
    • Popular in Hong Kong? ʳ
  • Zhengma
    • Supports simplified and traditional letters
  • Pinyin
  • Dayi
  • Zhuyin / Bopomofo
    • Common in Taiwan, but not China
  • CKC: https://en.wikipedia.org/wiki/CKC_Chinese_Input_System
    • Uses https://en.wikipedia.org/wiki/Four-Corner_Method
    • Characters have numbers (maximum 4 digits) associated with them
    • Needs only numberkey pad
  • Biaoxingma
    • Not very popular now
    • Letters are assigned Latin letters resembling their constituent parts

More:

  • QuanPin
  • ShuangPin

Wubi

  • Some wubi codes lead to whole a sentence.

  • Root shape => type its key 4 times to get its character ʳ

  • Repeated root shape => if there is a letter which is a root shape repated n times, press root shape key n times

    • 从 is 人 twice
    • 鑫 is 金 thrice

Zones

  • 5 zones
Zone Keys Stroke Shape Sample shape
1 GFDSA Left-right (horizontal) 一
2 HJKLM Top-bottom (vertical) 丨
3 TREWQ Falling left 丿
4 YUIOP Falling right 丶
5 NBVCX Hook 乙
  • GFDSA: keys goes from right to left though stroke is from left to right?
  • HJKLM: Note that M is away from the rest of the letters in this zone
  • TREWQ: leftwards 'slash' from top
  • YUIOP: rightward slash from top
  • NBVCX: zig-zag shape
  • Space: Finish a character

Emacs

  • Install these packages: pyim, pyim-wbdict
  • Change input method with C-\ and select pyim

pyim:

  • pyim allows user defined dicts as well. Use pyim-dicts-manager to customize.
  • Default mode of pyim seems to be for pinyin
